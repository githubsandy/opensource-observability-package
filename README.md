# Observability Installer

A one-click, Ansible-based installer for open-source observability tools: Prometheus, Node Exporter, Loki, and Grafana.

## Quick Start

```bash
./scripts/run_installer.sh
```

## Requirements

- Ansible 2.9+
- Python 3.x
- Linux VM or server

## Tools Installed

- Prometheus
- Node Exporter
- Loki
- Grafana
